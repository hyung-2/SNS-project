**재잘재잘**은 재잘거린다는 뜻처럼 가볍게 일기장을 쓰듯이, 친구와 대화하듯이 사용할 수 있는 SNS입니다.



### 주요기능
- 회원가입
- 로그인
- 게시글 & 댓글 작성,수정,삭제
- 좋아요
- 친구검색 & 팔로우
- 프로필 수정
